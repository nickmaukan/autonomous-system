# 🎮 Game Dev AI - Research

Investigación sobre automatización del desarrollo de juegos usando IA.

## 📋 Descripción

Documentación y experiments sobre cómo usar IA para automatizar diferentes aspectos del desarrollo de juegos.

## 🛠️ Áreas de Investigación

- Generación procedural de niveles
- NPCs con IA conversacional
- Generación automática de assets
- QA automatizado con machine learning
