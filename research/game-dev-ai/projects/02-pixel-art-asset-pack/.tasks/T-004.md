## Sprite Specifications

```
KNIGHT PLAYER
├── Resolution: 32x32
├── Style: 16-bit medieval
├── Animations:
│   ├── idle      → 4 frames (breathing)
│   ├── walk      → 8 frames
│   ├── attack    → 6 frames
│   ├── hurt      → 4 frames
│   └── death     → 4 frames
└── Output: PNG + sprite sheet grid
```

```
✓ Idle: 4 frames, transparent background
✓ Walk: 8 frames, transparent background
✓ Attack: 6 frames, transparent background
✓ Hurt: 4 frames, transparent background
✓ Death: 4 frames, transparent background
✓ Sprite sheets 2x4 o 4x2 grid
✓ Metadata JSON creado
✓ QA: Consistencia visual verificada
```
