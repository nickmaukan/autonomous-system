## Description

Configurar herramientas necesarias para generar pixel art medieval:
- Leonardo AI (generación de sprites)
- Crear bucket de Google Drive para assets
- Verificar acceso a Aseprite

---

## Acceptance Criteria

```
✓ Cuenta de Leonardo AI creada y verificada
✓ API key guardada en lugar seguro
✓ Folder de Google Drive creado
✓ Aseprite instalado y funcionando (del proyecto anterior)
✓ Carpeta local de assets creada
```

---

## Subtasks

- [ ] 1. Crear cuenta en Leonardo AI (leonardo.ai)
- [ ] 2. Completar onboarding y verificar email
- [ ] 3. Generar API key (Settings → API)
- [ ] 4. Guardar API key en lugar seguro (1Password o similar)
- [ ] 5. Crear folder en Google Drive: "MedievalPack_Assets"
- [ ] 6. Crear folder local: ~/MedievalPack/
- [ ] 7. Verificar Aseprite funciona: `aseprite --version`

## Notes

```
HERRAMIENTAS ALTERNATIVAS SI LEONARDO FALLA:
- Midjourney ($10/mes)

- ComfyUI (local)

LEONARDO FREE TIER:
- 150 tokens/día
- Suficiente para ~30-50 sprites por día
- Ideal para empezar
```
