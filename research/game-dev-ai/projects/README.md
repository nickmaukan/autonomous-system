## 📂 Detalle de Proyectos

### 🏆 Alta Prioridad (Alto impacto + Buena viabilidad)

#### 01 - Pipeline de Sprites con LoRA
**Por qué:** Resuelve el problema principal de inconsistencias en sprites AI.
- Crea sprites consistentes para personajes
- Automation del proceso
- Base para asset packs
📁 [Ver proyecto](./01-sprite-pipeline-lora/PROYECTO.md)

#### 03 - Platformer Prototype
**Por qué:** Demuestra el pipeline completo + Portfolio piece.
- Juego jugable de 5 niveles
- Combina arte + código AI
- Publicable en itch.io
📁 [Ver proyecto](./03-platformer-prototype/PROYECTO.md)

#### 07 - Game Jam Toolkit
**Por qué:** Reutilizable para TODOS los game jams futuros.
- Templates + prompts + scripts
- 60%+ tiempo ahorrado
- Workflow optimizado
📁 [Ver proyecto](./07-game-jam-toolkit/PROYECTO.md)

### 💰 Potencial de Ingreso

#### 02 - Pixel Art Asset Pack (Medieval)
**Por qué:** Monetizable directamente.
- Venta en Unity Asset Store / itch.io
- $200-1000 recurrentes por pack
- Skills transferibles

#### 06 - 3D Props Collection
**Por qué:** Nicho menos saturado que sprites 2D.

## 💡 Tips para Ejecutar

1. **Empieza con el Proyecto 01** - Es la base para muchos otros
2. **Paraleliza cuando puedas** - Arte AI + código pueden avanzar simultáneamente
3. **Documenta mientras avanzas** - Los learnings valen tanto como el código
4. **Itera en lugar de perfectionar** - Primeros drafts imperfectos > nunca empezar
5. **Pide feedback temprano** - Reddit, Discord, itch.io comments
