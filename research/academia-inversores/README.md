# 📈 Academia Inversores - Stock Analyzer

Script de análisis del mercado de valores para inversores.

## 📋 Descripción

Herramienta de screener y análisis técnico para identificar oportunidades de inversión en el mercado de acciones.

## 🛠️ Tecnologías

- **Python 3.x**
- **yfinance** - Datos de Yahoo Finance
- **pandas** - Análisis de datos
- **TA-Lib** - Indicadores técnicos

## 📦 Instalación

```bash
pip install yfinance pandas
```

## 🚀 Uso

```bash
python academia_analyzer.py
```

## 📊 Features

- Screener de acciones
- Análisis técnico automático
- Indicadores: RSI, MACD, SMA
- Generación de reportes
