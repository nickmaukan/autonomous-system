# 🚀 DEPLOY: Landing Page - 2 minutos

## Opción 1: Netlify (RECOMENDADO - más fácil)

1. Ve a: https://app.netlify.com/drop
2. Arrastra el archivo `index.html` (o el ZIP)
3. Listo! Te da un URL como: `random-name-123.netlify.app`
4. Comparte ese URL

## Opción 2: Vercel

1. Ve a: https://vercel.com/new
2. Click "Import Project"
3. Sube el folder con `index.html`
4. Deploy automático

## Opción 3: GitHub Pages

1. Crea repo en GitHub ( público)
2. Sube `index.html`
3. Settings → Pages → Deploy from main branch
4. URL: `tuusername.github.io/nombre-repo`

## Opción 4: Surge.sh (si tienes cuenta)

```bash
cd ~/AutonomousSystem/agency
surge . automatizamas-restaurantes.surge.sh
```
