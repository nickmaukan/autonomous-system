# 🔄 GUÍA 3: Sistema Cross-Posting IG → FB

**Tiempo de setup:** 30 minutos  
**Tiempo por post:** 5 minutos  
**Impacto:** Alto (duplica alcance sin doble trabajo)

---

## CONCEPTO

### Flujo:
```
IG Post → Adaptar caption → FB Post
     ↓
   Reel → Descargar → FB Reel
     ↓
   Stories → Screenshot → FB Stories
```

---

## MÉTODO 1: Manual (Gratis)

### Paso 1: Publicar en Instagram
1. Crear post en Instagram normalmente
2. Usar caption optimizado para IG
3. Agregar hashtags

### Paso 2: Copiar para Facebook
1. Copiar el caption del post
2. **Eliminar hashtags** (Facebook no los usa igual)
3. **Agregar @mentions** en lugar de hashtags
4. **Agregar CTA diferente** para FB

### Paso 3: Ajustes de Caption
```
ANTES (IG):
---
5 reglas de diseño que nunca fallan 👇
1. La luz natural es tu mejor aliada ☀️
2. Menos es más
3. Los neutrales son timeless
...

#ArquitecturaEcuador #Interiorismo #DiseñoDeInteriores

DESPUÉS (FB):
---
5 reglas de diseño que nunca fallan 👇

1. La luz natural es tu mejor aliada ☀️
2. Menos es más
3. Los neutrales son timeless
...

👇 COMENTA "DISEÑO" para más tips

👉 Síguenos en @entramados_estudio78

#Arquitectura #Interiorismo #Diseño #ArquitecturaEcuador
```

### Cambios Clave para FB:
- ❌ Eliminar hashtags específicos de IG
- ✅ Agregar @mentions (no funciona igual en FB)
- ✅ CTA diferente: "Comenta X" > "Guarda"
- ✅ Agregar link a IG en descripción
- ✅ Usar hashtags más generales

## MÉTODO 2: Later/Buffer (Semi-automático)

### Setup:
1. Crear cuenta en https://later.com (freemium)
2. Conectar Instagram y Facebook
3. Programar posts

### Ventajas:
- Programar posts para ambas plataformas
- Visual calendar
- Best time to post suggestions
- Hashtag sets guardados

### Cómo funciona:
1. Crear post en Later
2. Seleccionar "Instagram + Facebook"
3. Auto-adapta caption para cada plataforma
4. Programar fecha/hora
5. Publica automáticamente

## MÉTODO 3: Meta Business Suite (Oficial)

### Setup:
1. Ir a: https://business.facebook.com
2. Crear Business Account
3. Conectar página de Facebook
4. Conectar Instagram

### Funcionalidades:
- Programar posts para IG + FB
- Ver métricas combinadas

### Cómo programar:
1. Meta Business Suite → Create Post
2. Seleccionar "Instagram + Facebook"
3. Subir imagen
4. Escribir caption (optimizado para FB)
5. Programar fecha/hora
6. Publicar

## CROSS-POSTING PARA STORIES

### IG Stories → FB Stories

1. **Screenshot** de la story (o guardar como imagen)
2. **Subir a FB Stories** como imagen
3. **Agregar texto adicional** para FB

### Template FB Story:
```
📱 Ver más contenido en @entramados_estudio78 (IG)

O buscanos en Facebook:
facebook.com/entramadosestudio
```

---

## CROSS-POSTING PARA REELS

### IG Reel → FB Reel

1. **Grabar/crear reel en IG** (puede ser solo para FB)
2. **O:** Grabar video vertical (9:16)
3. **Subir a IG Reels**
4. **Subir mismo video a FB Reels**

### Diferencias:
| Aspecto | IG Reel | FB Reel |
|---------|---------|---------|
| Duración | 90 seg | 60 seg |
| Música | Trending | Trending |
| Caption | + hashtags | + CTA + @ |
| Cover | Imagen attractiva | Texto + logo |

### Para Re-exportar:
1. Descargar reel de IG (si está permitido)
2. O usar video original
3. Editar en CapCut/Canva para formato FB
4. Subir a FB Reels

## PLATAFORMAS ADICIONALES

### Pinterest
1. **Crear account** en pinterest.com
2. **Conectar** con IG (opcional)
3. **Crear Pin** con:
   - Imagen vertical (2:3 ratio)
   - Título optimizado SEO
   - Descripción con keywords
   - Link a website o IG

### TikTok
1. **Descargar reel de IG** (si es apropiado)
2. **Re-editar** para TikTok (formato, texto)
3. **Subir a TikTok** con:
   - @mention en caption
   - Hashtags #fyp #architecture #interiordesign
   - CTA para seguir
