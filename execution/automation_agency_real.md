### AUTOMATIZACIONES EXACTAS A IMPLEMENTAR:

#### 1. BOT DE MENÚ
```
Trigger: Cliente escribe "hola" o inicia chat
Flow:
- Mostrar mensaje de bienvenida
- Mostrar categorías del menú
- Cliente selecciona categoría
- Mostrar platos de esa categoría
- Cliente selecciona plato
- Preguntar cantidad
- Agregar al carrito
- Preguntar si quiere más
- Si no, mostrar resumen
- Pedir confirmación de datos (nombre, dirección, teléfono)
- Confirmar pedido
- Enviar notificación al restaurante
- Guardar en Google Sheets
- Enviar confirmación al cliente
```

#### 2. SEGUIMIENTO DE ESTADO
```
Trigger: Restaurante actualiza estado del pedido
Flow:
- Restaurante escribe código en WhatsApp
- Sistema actualiza estado
- Cliente recibe notificación automática:
  - "Tu pedido está siendo preparado"
  - "Tu pedido está listo"
  - "Tu pedido ha sido entregado"
```

#### 3. RECORDATORIOS Y FEEDBACK
```
Trigger: 30 min después de confirmado "entregado"
Flow:
- Enviar mensaje: "Cómo te pareció el pedido? Te agradeceríamos mucho una opinión"
- Si responde, guardar feedback
```
